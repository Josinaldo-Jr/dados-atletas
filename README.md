# Athlete Data Processing System

A concise JavaScript application developed to process athlete data in a gymnastics competition context.  
The system calculates and displays the athlete’s **category**, **BMI (Body Mass Index)**, **valid average score**, and other relevant information based on predefined rules. Built for Certification Project 2 as part of the DEVstart training program.

## Features

- Receives athlete personal data (name, age, weight, height)
- Receives five judges' scores
- Automatically sorts the scores
- Discards the highest and lowest scores
- Calculates the valid average score
- Calculates the athlete’s BMI
- Determines the athlete’s competition category
- Displays all results in the console
